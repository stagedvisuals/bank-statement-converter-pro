# Bank Statement Converter

Een eenvoudige tool om PDF bankafschriften om te zetten naar CSV of Excel.

## Deploy op Vercel

1. Fork deze repository naar je eigen GitHub account
2. Ga naar [vercel.com](https://vercel.com) en log in met GitHub
3. Klik "New Project" en selecteer deze repository
4. Klik "Deploy"
5. Je site is live binnen 2 minuten!

## Features

- Converteer ING, Rabobank, ABN AMRO en bunq afschriften
- Exporteer naar CSV of Excel
- 100% privacy - geen data opslag
- Gratis voor 5 conversies, daarna €5 per stuk of €29/maand onbeperkt
